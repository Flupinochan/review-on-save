# review-on-save README

In Development...

VSCode extension that performs code review using AI on file save

![preview.png](./img/preview.png)

> ⚠️ **Required**: This extension requires one of the following AI services:
> 
> - **Ollama**: Download and install from [ollama](https://ollama.com/download)
> - **GitHub Copilot**: Active subscription to [GitHub Copilot](https://marketplace.visualstudio.com/items?itemName=GitHub.copilot) required

## Overview
This extension automatically performs code review using generative AI when you save files, helping improve code quality and catch potential issues.
